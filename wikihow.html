How to Load a Project

To load a project into the website, follow these steps:

  1. Ensure your project is in PDF format.
  2. Upload the PDF file to the projects folder.
  3. Edit the projects.json file to include the new project details:
    • name: The name of the project.
    • url: The path to the PDF file.
    • description: A brief description of the project.
  4. Save the changes and refresh the projects page to see the new project in the carousel.
